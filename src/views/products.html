Despliegue de cartas pokemons

Esta aplicación recolecta información desde la API de pokemon.

En caso de que no haya una ruta valida muestra un error 404

Se utilizaron las tecnologías siguientes:

  • -Webpack (plugins, loaders)
  • -CSS (BOOTSTRAP y un ligero toque de SCSS)
  • -NPM
  • -HTML
  • -JavaScript
